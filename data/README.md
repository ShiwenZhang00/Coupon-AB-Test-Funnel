# Data Directory

The data directory contains various datasets and documentation files pertinent to the analysis of the Coupon AB Test Funnel project. Below is a detailed description of the files included in this directory:

## Coupon_Impact.csv
This file contains the results of the coupon impact analysis. It includes data on user interactions, coupon usage, and the overall effectiveness of the coupons in driving conversions.

### Key Columns:
- **user_id**: Unique identifier for each user.
- **coupon_code**: The code of the coupon used.
- **conversion**: Indicates whether the coupon led to a conversion (1 for yes, 0 for no).
- **timestamp**: Date and time of the user's interaction.

## Data readme.txt
This is a documentation file that provides an overview of the datasets included in the data directory. It explains the structure of each dataset and the methodology used for data collection and processing.

## Variables.txt
In this file, you will find a comprehensive list of the variables used across the datasets along with their definitions and data types. This serves as a reference for data analysts to understand the context and format of the data present in this directory.
